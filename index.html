Gebruik

1

Installeer het Package

Gebruik een van de bovenstaande installatiemethoden om KdG-Kiosk te installeren.

2

Configureer de Kiosk

Start de setup wizard vanuit het applicatiemenu ("Setup Kiosk"). Configureer:

  • Standaard browser (Chromium/Firefox)
  • Start-URL
  • Whitelist van toegestane websites
  • Exit sneltoets (standaard: Alt+5)
3

Start de Kiosk

Start KdG-Kiosk vanuit het applicatiemenu of voer uit:

kdg-kiosk
4

Afsluiten

Gebruik de geconfigureerde exit sneltoets (standaard Alt + 5) om de kiosk te stoppen.
